래퍼런스 자료형 : 클래스 , 인터페이스 .배열 

인터페이스(interface)
1. 인터페이스는 추상클래스의 일종이다 
2. 인터페이스의 모든 메서드는 추상메서드이다  (내용없는 메서드 =추상 메서드)
3. 인터페이스의 메서드 접근 제한자는 public이다 
	생략하더라도 public 이다 
4. 인터페이스 변수는 static final 이다 (변수를 상수화 하는것 )(값 변경 못한다 )
   static final을 생략하더라도 static final 이다 
   ex:
   static final int MAX = 100; //클래스에서 선언
   int MAX=100; // 인터페이스 에서 선언 
   값 변경 불가능 
 
 5. 생성자, static 메서드, private 메서드는 추상화 될 수 없다 
 6. 추상메서드는 상속받아서 오버라이딩 해야한다
 
 ※
 일반클래스 : 내용있는 메서드만 존재
 추상클래스 : 내용 있는 메서드도 있고, 추상 메서드도 함께 존재 한다  
 인터페이스 : 모든 메서드가 추상 메서드 이다
