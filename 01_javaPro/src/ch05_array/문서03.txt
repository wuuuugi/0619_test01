배열 선언 형식

자료형[] 배열명;
자료형 배열명[];
배열명 = new 자료형[요소갯수];
ex: int a[];  ,   a =new int[3] 

배열선언 형식2 
자료형 배열명[] =new 자료형 [요소갯수]; //선언과 동시에 생성 
ex: int a[] =new int[3]

배열선언 형식3 
 선언과 동시에 초기값을 준다 
 자료형 배열명[]= {초기값, 초기값,,,};
 ex: int a[]= {10,20,30};
 
 stack영역: 정적변수가 할당 되고 (일반변수 선언)(매서드)
 heap영역  : 동적메모리를 할당한다 (new 객체생성,new 배열생성)
 가비지 컬렉션 : JVM의 Runtime Data Area에서 사용하지 않은 heap메모리의 내용을 삭제하는것
